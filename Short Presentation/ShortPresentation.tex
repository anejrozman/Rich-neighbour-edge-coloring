\section{Uvod}

In this paper we set out to analyse an open conjecture in a modern graph theory problem known as rich-neighbour edge coloring.


\begin{definicija}
    In an edge coloring, an edge $e$ is called $rich$ if all edges adjacent to $e$ have different colors. An edge coloring is 
    called a $rich\text{-}neighbour \ edge \ coloring$ if every edge is adjacent to some rich edge.
\end{definicija}

\begin{definicija}
    $X'_{rn}(G)$ denotes the smallest number of colors for which there exists a rich-neighbour edge coloring.
\end{definicija}

\begin{conjecture}
    For every graph $G$ of maximum degree $\Delta$, $X'_{rn}(G) \leq 2\Delta - 1$ holds.
\end{conjecture}

\begin{primer}
    Let's take a look at the Petersen graph and an example of a rich-neighboiur edge coloring.
    \begin{center}
        \begin{tikzpicture}[every node/.style={draw,circle,very thick}]
            \graph[clockwise, radius=3cm] {subgraph C_n [n=5,name=A] };
            \graph[clockwise, radius=1.5cm] {subgraph I_n [n=5,name=B] };
        
            \foreach \i [evaluate={\j=int(mod(\i+2+4,5)+1)}]% using Paul Gaborit's optimisation
               in {1,2,3,4,5}{
              \draw (A \i) -- (B \i);
              \draw (B \j) -- (B \i);
            }
        \end{tikzpicture}
    \end{center}
    We can see that for the Petersen graph (which is 3-regular) $X'_{rn}=5 \leq 5$.
\end{primer}
